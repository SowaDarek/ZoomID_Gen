How to use:
1. Download one of the versions.
2. Open CMD and type "cd YourLocationWhereThePythonCodeIsLocated (ex. cd C:\Admin\Users\Downloads\)".
3. Type the version/s you selected, ex. ZoomID_GenV1, V2, V3 and add ".py" to the end of it.
4. Now you should have it working and running!

VERSIONS:

V1: Initial release. Generates 20 Zoom ID's.
V2: Asks user how many ID's to generate. (ex. "10")
V3: Added Zoom ID Validator. (checks if the ID is valid so it can't be random numbers or letters.)
